## Random Quote API (Rails)

Legacy Rails API for serving quote data.

### Status
- Maintained in **minimal/legacy mode**
- Original Heroku URL in old README is no longer guaranteed active

### Stack
- Ruby `2.5.1`
- Rails `5.2.1`

### Local run (legacy)
```bash
bundle install
rails db:setup
rails s
```
